# SRT → Audio Converter

Convert subtitle (.srt) files to spoken audio using **Microsoft Edge TTS** — free, no API key required.

## Setup

```bash
yarn install
```

## Usage

```bash
yarn dev
```

1. Drop or select an `.srt` file
2. Pick a voice by locale and name
3. Click **Start Conversion**
4. Download the MP3

## Deploy to Vercel

```bash
npx vercel
```
